    • The Mandelbrot and Multibrot Sets

    The Mandelbrot set is created by iterating infinitely the function $f(z) = z^2 + c$ ( $z_o=0$) for different values of c and checking for convergence. If a complex number $c$ converges under these conditions it is considered to be within the Mandelbrot set. However a generalization of the Mandelbrot set exists called the Multibrot set where instead the function is $f(z) = z^d+c$. This page allows you to view how the plot changes for different $d$ values and explore the Multibrot set.
